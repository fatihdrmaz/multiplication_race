# Assets Notları

Bu klasördeki görsel dosyalar için:

## İkon Dosyaları

Oyun için özel ikonlar oluşturmak istersen, şu özelliklere sahip olmalı:

### icon.png
- **Boyut**: 1024x1024 px
- **Format**: PNG (şeffaf arka plan)
- **İçerik**: Gülen araba yüzü + çarpım sembolü (×)
- **Stil**: Parlak renkler, kalın konturlar, basit şekiller

### splash.png
- **Boyut**: 1242x2436 px (dikey)
- **Format**: PNG
- **İçerik**: "Vroom - Çarpım Yarışı" logosu, araba grafikleri
- **Arka Plan**: Mavi gradient (#4FC3F7)

### adaptive-icon.png (Android)
- **Boyut**: 1024x1024 px
- **Format**: PNG (şeffaf)
- **İçerik**: icon.png ile aynı ama kenarlardan 20% boşluk bırak

### favicon.png (Web)
- **Boyut**: 48x48 px
- **Format**: PNG
- **İçerik**: icon.png'nin küçültülmüş hali
